Passing Arguments to a Function

Aim

This virtual lab aims to demonstrate the various ways arguments can be passed to a function, including positional arguments, keyword arguments and default arguments.

Theory & Applications

This section covers the fundamental theories and applications of the concept.

Key Concepts
  • Function: A function is a block of reusable code designed to perform a specific task.
  • Parameter: A parameter is a variable in the function definition that acts as a placeholder for the argument. It defines what type of data a function expects.
  • Argument: An argument is the value passed to a function while calling it. It is the input the function uses to perform its task.
    • There are different argument types, making functions more flexible and reusable. Some argument types in python are: Default arguments, Keyword arguments, and Positional arguments.

Types of Arguments

  • Default Arguments:
    • Default arguments in a function are assigned predefined values during the function declaration. These default values are used if the caller does not provide a corresponding argument during the function call. The default value is assigned using the assignment(=) operator in the form keywordname=value.
    Syntax:
    def function_name(var1, var2=def_val2, var3=def_val3) [Function Declaration]
        function_name(val1, val2)     [Function Call]

    Coding
    Define a function and specify the parameters. Assign default values to the parameters.
    Call the function without any argument values: in this case, the function will calculate the BMI using the default values.

    Call the function with one or both arguments: the function will use the given values instead of the default values and compute the result.

    In this case, the BMI is calculated using the arguments passed while calling the function.

  • Keyword Arguments
    • Keyword arguments refer to arguments that are passed using parameter names during function call. The order of keyword arguments does not affect the output, as long as the logic of the code is correct. This ensures that the function works as intended regardless of the argument order.
    Syntax:
    def function_name(var1, var2, var3) [Function Declaration]
        function_name(var1=value1, var2=value2, var3=value3)     [Function Call]
    Coding
    Define a function and specify the parameters.
    Call the function with keyword arguments: Specify arguments using the parameter names defined in the function.
    This allows flexibility in the order of values. In both cases, the output is correct.

  • Positional Arguments
    • Positional Arguments are arguments that must be passed in a specific order when calling a function. If the order of arguments passed differs from the order of parameters defined, it will result in an unexpected output.
    Syntax:
    def function_name(var1, var2, var3) [Function Declaration]
        function_name(value1, value2, value3)     [Function Call]
    Coding
    Define a function and specify the parameters.
    Call the function with positional arguments: When calling the function, provide arguments in the order that matches the parameter sequence defined in the function. Calling the function in the wrong order will result in an incorrect output. As shown in the second function call statement.
    Hence, the position of the arguments matters.




Applications in Real World
  1. Data analysis: Arguments help in data cleaning, transformation and analysis.
  2. Game Development: Allows developers to create more flexible and dynamic gameplay mechanics.
  3. Automation Scripts: Arguments allow automation scripts to be more adaptable to different scenarios.
Procedure

This section provides the steps and procedure for implementing the concept discussed above.

Code

Example program


#BMI Calculator
# Default Arguments 
def bmi_calc(weight=45, height=1.4):
    bmi = weight / (height ** 2)
    return bmi

result = bmi_calc() # assigns defalt values
result2 = bmi_calc(60, 1.6) # assigns 60 to weight and 1.6 to height
print("BMI:", result)
print("BMI:", result2)

# Keyword Arguments
def bmi_calc(weight, height):
    bmi = weight / (height ** 2)
    return bmi

result = bmi_calc(weight = 55, height = 1.6) # arguments passed with keywords
result2 = bmi_calc(height = 1.6, weight = 55)
print("BMI:", result)
print("BMI:", result2)


# Positional Arguments
def bmi_calc(weight, height):
    bmi = weight / (height ** 2)
    return bmi

result = bmi_calc(60, 1.63) # correct order of arguments
result2 = bmi_calc(1.63, 60) # incorrect order, value of height given before weight
print("BMI:", result)
print("BMI:", result2)
